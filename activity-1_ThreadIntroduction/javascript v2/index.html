Thread Intro with Typescript

This is the intro thread assignment using typescript. Typescript doesn't support threading, but does support web workers, which are similar, but they don't share memory with the main thread. I've attempted to emulate the behavior of threads despite this.

The console shows what the main thread is printing. The Workers don't have access to the console.

You can create a thread as you would in java, and calling 'start' on it. You can use Log.log('')
